# JTBD (Jobs To Be Done)

## プロダクト名: グロースパック for LINE デジタル会員証

## メインジョブ

When 店舗でお買い物をしていてレジで会員証を提示したい時
I want to アプリをダウンロードせずにすぐに会員証を表示したい
So that 待ち時間なくスムーズにポイントを貯められる

## 関連ジョブ

### ジョブ1: 仮会員としてすぐにポイントを貯める

When 店頭で初めて会員になろうとした時
I want to QRコードを読み取るだけで仮会員証を取得したい
So that 面倒な入力作業なしで今すぐポイントを貯め始められる

### ジョブ2: 本会員登録でポイントを使う

When 仮会員としてポイントが貯まってきた時
I want to 本会員登録を完了してポイントを使えるようにしたい
So that 貯めたポイントで割引を受けられる

### ジョブ3: 本会員登録の特典を獲得する

When 仮会員から本会員になるか迷っている時
I want to 本会員登録で得られる特典を知りたい
So that 登録するメリットを理解して判断できる

### ジョブ4: 会員ランクアップ

When 継続的に利用している時
I want to 自分のランクと次のランクまでの進捗を把握したい
So that モチベーションを持って買い物を続けられる

## ジョブの優先順位

| 優先度 | ジョブ | 理由 |
|--------|--------|------|
| 1 | 仮会員証取得 | レジ前での離脱を防ぐ最速の会員化 |
| 2 | 本会員登録 | ポイント利用解放とCRM強化 |
| 3 | 登録特典獲得 | 本会員化のインセンティブ |
| 4 | ポイント確認 | 購買意欲を高める補助機能 |
