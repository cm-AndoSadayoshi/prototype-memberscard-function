# ユーザーストーリーマッピング

## プロダクト名: グロースパック for LINE デジタル会員証

---

## ユーザーペルソナ

| 項目 | 内容 |
|------|------|
| 名前 | 田中 花子（35歳） |
| 属性 | 主婦、週2-3回スーパーで買い物 |
| 目標 | 効率よくポイントを貯めてお得に買い物したい |
| 課題 | 財布のカードが多すぎて会員証を出すのが面倒 |

---

## ユーザーアクティビティ（大きな流れ）

```
[QR読み取り] → [仮会員証表示] → [本会員登録] → [本会員証表示]
```

---

## ストーリーマップ

### アクティビティ1: QR読み取り（仮会員登録）

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| QRコード読み取り | As a 新規顧客, I want to 店頭のQRコードを読み取るだけで仮会員になりたい, So that 面倒な入力なしで今すぐポイントを貯められる | MVP |
| LINE連携認証 | As a 新規顧客, I want to LINEアカウントで自動認証したい, So that パスワード管理が不要になる | MVP |

### アクティビティ2: 仮会員証表示

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| バーコード表示 | As a 仮会員, I want to ワンタップでバーコードを表示したい, So that レジでポイントを貯められる | MVP |
| ポイント残高確認 | As a 仮会員, I want to 貯まったポイントを確認したい, So that 本会員登録のモチベーションになる | MVP |
| 本会員登録への誘導 | As a 仮会員, I want to 本会員登録のメリットを知りたい, So that ポイントを使えるようになる方法がわかる | MVP |

### アクティビティ3: 本会員登録

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| 会員情報入力 | As a 仮会員, I want to 必要最小限の情報だけ入力したい, So that 素早く本会員になれる | MVP |
| 登録完了クーポン獲得 | As a 新規本会員, I want to 登録完了時にクーポンを受け取りたい, So that すぐにお得な買い物ができる | MVP |
| ポイント利用解放 | As a 新規本会員, I want to 貯めたポイントを使えるようになりたい, So that 割引を受けられる | MVP |

### アクティビティ4: 本会員証表示

| タスク | ユーザーストーリー | 優先度 |
|--------|-------------------|--------|
| 本会員バーコード表示 | As a 本会員, I want to ワンタップでバーコードを表示したい, So that ポイントを貯める・使うができる | MVP |
| ランク確認 | As a 本会員, I want to 自分の会員ランクを確認したい, So that 特典内容を把握できる | MVP |
| クーポン確認 | As a 本会員, I want to 保有クーポンを確認したい, So that お得に買い物できる | 後回し |

---

## MVPスコープ

### MVP機能一覧

1. QRコード読み取り→仮会員証即発行
2. 仮会員証表示（バーコード/QR）※ポイント付与のみ
3. ポイント残高表示（仮会員でも確認可能）
4. 本会員登録フォーム（必要最小限の入力）
5. 登録完了クーポン進呈（500円OFFなど）
6. 本会員証表示（バーコード/QR）※ポイント利用可能
7. 会員ランク・進捗表示

### 後回し機能

1. ポイント履歴詳細
2. クーポン一覧管理
3. オフライン対応
4. 会員情報編集
