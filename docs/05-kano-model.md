# 狩野モデル（機能優先順位）

## プロダクト名: グロースパック for LINE デジタル会員証

---

## 品質分類

### 当たり前品質 (Must-be) - 必須

ないと不満だが、あっても満足度は上がらない機能。

| 機能 | 説明 | MVP判断 |
|------|------|---------|
| 仮会員証表示（バーコード/QR） | QR読み取りで即発行、ポイント付与用 | ✅ 必須 |
| 本会員証表示（バーコード/QR） | 本会員登録後、ポイント利用可能 | ✅ 必須 |
| ポイント残高表示 | 現在のポイント残高を確認できる | ✅ 必須 |
| LINE認証 | LINEアカウントでの自動認証 | ✅ 必須 |
| 本会員登録フォーム | 仮会員から本会員への移行 | ✅ 必須 |

---

### 一元的品質 (Performance) - 選択的

あればあるほど満足度が上がる機能。

| 機能 | 説明 | MVP判断 |
|------|------|---------|
| 表示速度 | 会員証画面の読み込み速度 | ⚪ 検討 |
| ポイント履歴 | 過去のポイント獲得・使用履歴 | ⚪ 検討 |
| クーポン一覧 | 保有クーポンの一覧と利用 | ⚪ 検討 |

---

### 魅力的品質 (Delighter) - 1つに集中

なくても不満はないが、あると感動する機能。

| 機能 | 説明 | MVP判断 |
|------|------|---------|
| **登録完了クーポン** | 本会員登録完了時に500円OFFクーポンを即時進呈。登録のインセンティブとなり、初回購買を促進 | ⭐ MVP採用 |
| ランクアップ演出 | ランクが上がった時の祝福アニメーション | 後回し |
| パーソナライズ提案 | 購買履歴に基づくおすすめ | 後回し |

---

## MVP機能サマリー

### 採用機能

| 品質タイプ | 機能 |
|------------|------|
| 当たり前品質 | 仮会員証表示、本会員証表示、ポイント残高、LINE認証、本会員登録フォーム |
| 魅力的品質（1つ） | 登録完了クーポン（500円OFF） |

### 見送り機能

| 機能 | 理由 |
|------|------|
| ポイント履歴詳細 | MVP後の拡張として対応 |
| クーポン一覧機能 | 登録完了クーポンのみMVPに含む |
| ランクアップ演出 | 魅力的だが登録クーポンに集中 |
| オフライン対応 | 技術的複雑さが高い |

---

## 仮会員→本会員 転換のインセンティブ設計

| インセンティブ | 内容 | 効果 |
|---------------|------|------|
| 登録完了クーポン | 500円OFFクーポン即時進呈 | 即時的なメリットで登録を後押し |
| ポイント利用解放 | 本会員のみポイント利用可能 | 貯まったポイントを使いたい欲求を刺激 |
| ランク制度参加 | 本会員のみランクアップ対象 | 継続利用のモチベーション |
